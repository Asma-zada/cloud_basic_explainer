## Project title
Cloud Basic Explainer

## Use case
This tools answers basic questions about cloud computing in simple, non-technical language. It helps business owners and beginners understand what the cloud is, how it works, and whether they should use it.

## Target user
1. Business owner deciding if they need cloud services.
2. Students and beginners learning cloud concepts.
3. Non-technical people who find cloud terms confusing.

## Examples Question Users May Ask
1. What is cloud computing?
2. What is the diffrence between AWS,Azure, and Google Cloud? 
3. What do SaaS, IaaS, and PaaS means?
4. Is my data safe in the cloud?
5. How much does cloud computing cost?
6. Why should business use the cloud?
7. What is the difference between cloud and storing data on my own computer?
